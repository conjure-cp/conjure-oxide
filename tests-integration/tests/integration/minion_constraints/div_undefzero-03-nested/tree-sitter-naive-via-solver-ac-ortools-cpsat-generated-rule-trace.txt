Model before rewriting:

find x: int(5..20)
find y: int(0..5)
find z: int(0..6)

such that

((x / (y / z)) = 10)

--

(y / z), 
   ~~> div_to_bubble ([("Bubble", 6000)])
{SafeDiv(y, z) @ (z != 0)}

--

(x / {SafeDiv(y, z) @ (z != 0)}), 
   ~~> bubble_up ([("Bubble", 8800)])
{(x / SafeDiv(y, z)) @ (z != 0)}

--

({(x / SafeDiv(y, z)) @ (z != 0)} = 10), 
   ~~> bubble_up ([("Bubble", 8800)])
{((x / SafeDiv(y, z)) = 10) @ (z != 0)}

--

{((x / SafeDiv(y, z)) = 10) @ (z != 0)}, 
   ~~> expand_bubble ([("Bubble", 8900)])
and([((x / SafeDiv(y, z)) = 10),(z != 0);int(1..)])

--

and([((x / SafeDiv(y, z)) = 10),(z != 0);int(1..)]), 
   ~~> constant_evaluator ([("Constant", 9001)])
((x / SafeDiv(y, z)) = 10),
(z != 0)

--

(x / SafeDiv(y, z)), 
   ~~> div_to_bubble ([("Bubble", 6000)])
{SafeDiv(x, SafeDiv(y, z)) @ (SafeDiv(y, z) != 0)}

--

({SafeDiv(x, SafeDiv(y, z)) @ (SafeDiv(y, z) != 0)} = 10), 
   ~~> bubble_up ([("Bubble", 8800)])
{(SafeDiv(x, SafeDiv(y, z)) = 10) @ (SafeDiv(y, z) != 0)}

--

{(SafeDiv(x, SafeDiv(y, z)) = 10) @ (SafeDiv(y, z) != 0)}, 
   ~~> expand_bubble ([("Bubble", 8900)])
and([(SafeDiv(x, SafeDiv(y, z)) = 10),(SafeDiv(y, z) != 0);int(1..)])

--

and([(SafeDiv(x, SafeDiv(y, z)) = 10),(SafeDiv(y, z) != 0);int(1..)]),
(z != 0), 
   ~~> constant_evaluator ([("Constant", 9001)])
(SafeDiv(x, SafeDiv(y, z)) = 10),
(SafeDiv(y, z) != 0),
(z != 0)

--

Final model:

find x: int(5..20)
