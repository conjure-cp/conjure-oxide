How to use this tool

  • Please, allow up to a minute for the table to load. The script generates a static HTML file which is quite big, so it may take a while for your browser to render all of it.
  • Click on table headers to sort
  • Shift+Click multiple headers to sort by multiple values
  • Use the search box in the top right corner of the table for search
  • Use the setting in the top left corner to set the number of rows to display
  • Use buttons at the bottom of the table to navigate between pages
  • See environment variables in the GitHub action file for configuration
