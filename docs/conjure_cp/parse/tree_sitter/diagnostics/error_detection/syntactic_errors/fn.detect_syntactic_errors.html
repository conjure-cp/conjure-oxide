pub fn detect_syntactic_errors(source: &str) -> Vec<Diagnostic>
Expand description

Detects syntactic issues in the essence source text and returns a vector of Diagnostics.

This function traverses the parse tree, looking for missing or error nodes, and generates diagnostics for each. It uses a DFS and skips children of error/missing nodes to avoid duplicate diagnostics. If the source cannot be parsed, a diagnostic is returned for that.
