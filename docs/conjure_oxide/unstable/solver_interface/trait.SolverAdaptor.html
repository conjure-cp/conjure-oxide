A SolverAdaptor provide an interface to an underlying solver. Used by Solver.

Required Methods§

source

fn solve( &mut self, model: Self::Model, callback: fn(_: HashMap<String, String>) -> bool, _: Internal ) -> Result<ExecutionSuccess, ExecutionFailure>

Run the solver on the given model.

Implementations of this function MUST call the user provided callback whenever a solution is found. If the user callback returns true, search should continue, if the user callback returns false, search should terminate.

source

fn solve_mut( &mut self, model: Self::Model, callback: fn(_: HashMap<String, String>, _: Self::Modifier) -> bool, _: Internal ) -> Result<ExecutionSuccess, ExecutionFailure>

Run the solver on the given model, allowing modification of the model through a ModelModifier.

Implementations of this function MUST return OpNotSupported if modifying the model mid-search is not supported. These implementations may also find the NotModifiable modifier useful.

As with solve, this function MUST call the user provided callback function whenever a solution is found.
