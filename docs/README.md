# The Book!
This is the (in progress) complete documentation for Conjure Oxide, dubbed 'The Book'.

## To contribute
Make a folder relating to your project in the src directory, and populate it together with your team, or put your contribution where you think the it should go. 

Don't worry too much, as it can always be moved.

If you are unsure where to put your documentation, you can put it in the 'misc' directory.
